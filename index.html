EXPLICANDO PHPBB



O PHPBB
é um pacote para criação de fóruns totalmente personalizável e em código aberto. Possui uma interface amigável ao usuário, simples e com um painel de administração avançado, contando também com uma seção de perguntas freqüentes bem útil. Baseado em linguagem PHP, o programa pode ser usado com seu sistema de administração de bancos de dados predileto.


FACILIDADE DE USO
A nova versão do programa está disponível também em português. Esse elemento configura um grande facilitador no processo de utilização do programa. Mas não só o idioma contribui para sua facilidade de uso. O envio de informações é tão simples como o envio de um e-mail e você pode fazer tudo de forma online.
